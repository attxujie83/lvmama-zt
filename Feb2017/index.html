RUN FOR FUN, RUN FOR HEALTH

由马里亚纳观光局（Marianas Visitors Authority）与北马里亚纳竞技联盟（Northern Marianas Athletic）共同主办的塞班马拉松已连续举办11年。今年的比赛项目包括：50公里、全程马拉松、半程马拉松、10公里。

塞班马拉松每年都会在最美的凤凰花季如期举办。晨光熹微，跑者从美国纪念公园出发，沿着海滨大道奔跑于塞班秀美的西海岸线上，将沿途热带美景尽收眼底，在工作人员与当地居民的喝彩中轻松完赛，尽情享受奔跑的乐趣。

官方认证 打造耳目一新的标准化国际赛事

塞班马拉松已成功取得国际田径联合会（IAAF）赛事资格认证。

塞班马拉松正式取得IAAF-AIMS的赛道认证。标志着马里亚纳群岛体育赛事的吸引力将提升至全新的高度，塞班马拉松如今已经成为包括波士顿马拉松在内的众多国际赛事的经营资格赛事。接下来，马里亚纳观光局将继续努力，申请成为AIMS的全资格成员，进而在未来通过AIMS的全球推广渠道，为塞班马拉松吸引更多的长跑爱好者。
